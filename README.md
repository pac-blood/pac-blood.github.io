# Pac-Blood

<p align="center"><img src="Pac blood 6.png" alt="Pac-Blood logo" width="200"></p>

<p align="center"><i>A Pac-Man-inspired 2D game made in Godot 4</i></p>

<p align="center">As a simple cell in a body, survive bacterias and win!</p>

## How to Play

1. Go to [https://pac-blood.github.io](https://pac-blood.github.io)

2. Press the `Play` button

3. Use the arrow keys to control the player cell

4. Survive a total of 3 rounds from bacteria that will spawn in random areas!

## Local Development

### Prerequisites

- A modern web browser

- A code editor

### How to Run

1. Open a terminal

2. Clone this repository with Git:

    ```bash
    git clone https://github.com/pac-blood/pac-blood.github.io.git
    cd pac-blood.github.io
    ```

3. Open your file explorer

4. Navigate to the `pac-blood.github.io` directory

5. Open `index.html` in your browser
